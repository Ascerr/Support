If you have problems with runing Rifbot on Realesta follow this steps:

1. Download RealestaOGL.exe
2. Repleace in Realesta folder.
3. Run game.
4. Run Rifbot.


If you have problems with runing Rifbot on Retrocores follow this steps:

1. Download client_ogl.xe
2. Go to Retrocores\data and repleace client_ogl.exe to new one.
3. Run game.
4. Run Rifbot.
